# bash的基本特性
## (1) 命令历史 ``` history ```

> HISTFILE: ~/.bash_history

显示历史中的命令
``` history # ``` 显示历史最近的#条命令

调用历史中的命令
``` !# ``` 重复执行第#条命令
``` !! ``` 重复上次的命令
``` !String ``` 执行String开头的命令

调用上一条指令的最后一个参数
``` !$ ``` 
``` ESC,. ```

控制命令历史的记录方式
